# SKUDSystem

Система управления доступом (СКУД).

## Функционал

- Авторизация
- Регистрация
- Роли пользователей:
  - Администратор
  - Исполнитель
  - Клиент
- Управление заявками
- Управление пользователями

## Архитектура

Проект реализован на:
- Java
- Android SDK
- XML-разметки
- Activity-based навигация

## Структура проекта

11 Activity
10 XML разметок
Полностью настроенный AndroidManifest
